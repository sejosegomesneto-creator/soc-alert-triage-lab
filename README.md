# SOC Alert Triage Lab

Cybersecurity laboratory simulating the triage process used by Security 
Operations Center (SOC) analysts.

This project demonstrates how security alerts can be classified based on 
severity and indicators of compromise.

## Objective

Simulate the workflow of a SOC Level 1 analyst when receiving alerts from 
monitoring tools.

## Triage Workflow

Alert → Analysis → Severity Classification → Recommended Action

## Technologies

Python
Log Analysis
Security Monitoring
